| Operator | Meaning              |
| -------- | -------------------- |
| `=`      | `x = 5`              |
| `+=`     | `x += 1` (x = x + 1) |
| `-=`     | `x -= 1`             |
| `*=`     | `x *= 2`             |
| `/=`     | `x /= 2`             |
| `//=`    | `x //= 2`            |
| `%=`     | `x %= 2`             |
